#   JIGSAW: an unstructured mesh generation package 
#------------------------------------------------------------

    JIGSAW is an unstructured mesh generator for 2-
    and 3-dimensional problems. 

    See EXAMPLE.JIG to run an example problem using
    JIGSAW.

    JIGSAW is a "restricted" Delaunay-refinement a-
    lgorithm, based (primarily) on ideas contained
    in the following references:

    [1] Darren Engwirda and David Ivers, Off-centre 
        Steiner-points for Delaunay-refinement on 
        curved surfaces. Computer Aided Design, Vo-
        lume 72, March 2016, Pages 157-171,  
        http://dx.doi.org/10.1016/j.cad.2015.10.007

    [2] Darren Engwirda, Voronoi-based Point-place-
        ment for Three-dimensional Delaunay-refine-
        ment, Proceedngs of the 24th International
        Meshing Roundtable, Procedia Engineering, 
        Volume 124, 2015, Page 330-342,
        http://dx.doi.org/10.1016/j.proeng.2015.10.143

    [3] Darren Engwirda, Locally-optimal Delaunay--
        refinement and optimisation-based mesh gen-
        eration. Ph.D. Thesis, School of Mathemati-
        cs and Statistics, University of Sydney, 2015,
        http://hdl.handle.net/2123/13148

    See the full-text articles for additional info-
    rmation and references.
